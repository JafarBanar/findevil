CaseTrace is a read-only DFIR agent for SIFT. This demo walks through a completed analysis of a reproducible NTFS image. The summary shows four retained findings, including suspicious script execution, likely payload delivery, persistence, and a YARA match. The iteration history shows the self-correction loop, where the first pass leaves corroboration gaps and the second pass gathers stronger timeline, registry, scheduled task, browser, and execution evidence. The findings and report then link each conclusion back to concrete forensic artifacts, while one unsupported credential-theft claim is explicitly blocked instead of being overstated.
